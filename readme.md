# Student Management App

A simple web application for managing student records with MongoDB.

## Features

- Add/Edit/Delete students with name and roll number
- View all students in real-time
- RESTful API endpoints
- Responsive UI

## Quick Start

### Prerequisites

- Node.js
- MongoDB

### Installation

```bash
npm install
```

### Run

```bash
npm run dev
```

Open `http://localhost:8000`

## Tech Stack

- Node.js & Express.js
- MongoDB & Mongoose
- HTML/CSS/JavaScript

## API Endpoints

- `GET /students` - Get all students
- `POST /addStudent` - Add student
- `PUT /students/:id` - Update student
- `DELETE /students/:id` - Delete student

## Usage

1. **Add Student**: Enter name and roll number, click "Add Student"
2. **Edit Student**: Click "Edit" button next to a student
3. **Delete Student**: Click "Delete" button to remove
